# 前言

欢迎来到招生宣传管理系统项目！这是一个使用Java语言、Spring Boot框架、Vue.js前端技术等开发的实战项目。此项目适用于计算机专业毕业设计，帮助你快速掌握项目开发流程和技能。以下是项目的详细介绍。

## 内容介绍

招生宣传管理系统旨在帮助学校或教育机构高效地管理招生宣传活动。系统主要功能包括：学生信息管理、招生计划管理、宣传资料管理、录取通知管理等。通过此系统，可以实现招生宣传的流程化、自动化，提高工作效率，降低人力成本。

## 技术介绍

本项目采用以下技术栈：

- **语言：Java**
- **使用框架：Spring Boot**
- **前端技术：JS、Vue、css3**
- **开发工具：IDEA/Eclipse**
- **数据库：MySQL 5.7/8.0**
- **数据库管理工具：phpstudy/Navicat**
- **JDK版本：jdk1.8**
- **Maven: apache-maven 3.8.1-bin**
- **前端环境：Node.Js 12\14\16**
